# Cierre de Inventario · Camelia Modern Mexican Cuisine

Aplicación Streamlit para documentar profesionalmente el cierre de inventario de Camelia.

## Accesos iniciales
- Administrador: `administrador` / PIN `5866`
- Camelia Robles: `camelia` / PIN `2026`

## Ejecución
```bash
pip install -r requirements.txt
streamlit run app.py
```

La base `camelia_inventory.db` se crea automáticamente. Descarga respaldos Excel frecuentes. Para almacenamiento permanente en la nube conviene migrar posteriormente a PostgreSQL/Supabase.
